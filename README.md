# checkThat

## Depencies

[jQuery (v1.6+)](https://jquery.com)

## Quick start

```html
<link rel="stylesheet" href="checkthat.min.css" />
<script type="text/javascript" src="checkthat.min.js"></script>
```

```javascript
//initialize plugin
$('target-input').checkThat();
//destroy plugin
$('target-input').checkThat('destroy');
```
